Patch di controllo v0.1 - Introduzione

Questa nuova versione della patch non richiede la versione PureData Extended ma la più semplice e stabile PureData Vanilla (v0.47).

Prima di avviare la patch occorre prima di tutto scaricare PureData Vanilla da questo link e installarla sul proprio sistema. Una volta fatto, scaricare questo archivio .zip e scompattarlo. la patch può essere avviata facendo doppio click sul file main.pd contenuto all'interno dell'archivio.

All'apertura la patch si presenta come mostrato nell'immagine seguente.

Nel creare la patch, considerato quanto detto negli incontri precedenti, abbiamo previsto che la sessione di Ableton Live sia composta di 5 tracce + 1, rispettivamente:

  1. prox
  2. baule
  3. portiera
  4. volante
  5. cambio
  6. cofano

Ognuna delle tracce è controllata da uno degli elementi della patch come indicato dalla seguente tabella:

PD elemento Live trk mappato su protocollo
prox slider prox volume OSC
baule toggle baule mute OSC
portiera toggle portiera mute OSC
volante toggle volante mute OSC
cambio toggle cambio mute OSC
cofano slider cofano a scelta (utilizzare la modalità Midi Map MODE di Ableton Live) MIDI CC 12

Come si vede la maggior parte dei controlli della patch pilotano Live utilizzando il protocollo OSC per cui non possono essere liberamente mappati. E' diverso il comportamento dello slider denominato cofano il quale, inviando messaggi di tipo MIDI sottoforma di Control Change #12, può essere mappato a piacere su qualsiasi parametro si preferisca.

Istruzioni

OSC in Ableton Live

Il plugin LiveOSC era stato in origine sviluppato per controllare Live tramite il monome. Il plugin può essere scaricato da questo sito o da questo link diretto. Di seguito un estratto della procedura di intallazione:
  1. Unzip the latest release which should give you a folder called 'trunk' containing a folder called 'LiveOSC'. The 'LiveOSC' folder should be moved to Ableton’s MIDI Remote Scripts folder;
  2. On OSX this is located in /path/to/ableton/Live.app -> Right click -> show package contents. Navigate to contents/app-resources/MIDI Remote Scripts;
  3. In Ableton goto Preferences, Midi Settings. Select 'LiveOSC' from the drop down list.
